## Верификация по документации

### Google Sheets API: insertDimension

- **DimensionRange**: `startIndex`, `endIndex` — 0-based, half-open `[start, end)`.
- **Вставка N строк**: `startIndex = X`, `endIndex = X + N` → вставка N строк начиная с индекса X.
- **Row 0** = заголовок, **Row 1** = первая строка данных (sheet row 2).

### Индексация в коде

| Сущность | Значение | Пояснение |
|----------|----------|-----------|
| `task.rowIndex` | 2 для первой строки данных | `parseRow(rows[i], i + 2)` → rowIndex = sheet row |
| `startRow0 = task.rowIndex` | 2 | Используется как startIndex для insert |
| Sheet row 2 | 0-based index 1 | Первая строка данных |
| Insert below row 2 | startIndex = 2 | Вставка после index 1 |
| Новые строки после insert | 0-based 2..30 | 29 строк |
| Range для values | A3:O31 | 1-based rows 3..31 |

**Вывод:** Формула `A${startRow0 + 1}:O${startRow0 + numRows}` даёт A3:O31 при startRow0=2, numRows=29 — корректно.

### valueInputOption: RAW

Во всех записях используется `RAW` — защита от formula injection (headline, keyword, status из AI/внешних источников).
